# 🐦 Flappy Bird Game (Python + Pygame)

## 📌 Overview

This is a simple Flappy Bird clone built using Python and Pygame.
The player controls a bird that must pass through pipes without colliding.

---

## 🎮 Features

* Smooth bird movement with gravity
* Jump mechanics using SPACE key
* Random pipe generation
* Collision detection
* Score tracking
* Game over and restart functionality

---

## 🛠️ Technologies Used

* Python 3.11
* Pygame

---

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/flappy-bird-game.git
cd flappy-bird-game
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the game

```bash
python flapping_bird.py
```

---

## 🎮 Controls

* SPACE → Jump
* R → Restart
